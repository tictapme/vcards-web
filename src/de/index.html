Fordern Sie eine personalisierte Demo an
01 Muss ich eine Anwendung installieren?

Nein, unsere Lösung ist völlig anwendungsfrei. Ihre vCard wird in der Cloud als URL gespeichert, die Sie auf Ihrem Desktop speichern können, um jederzeit einfach darauf zugreifen zu können.

02 Wer kann die Inhalte der Karten verwalten?

Nur der/die Administrator(en) haben Zugriff auf die zentrale Inhaltsverwaltung.

03 Kann ich vCards unter meiner eigenen Webdomäne verwenden?

Ja, Sie können das vCard-System unter einer eigenen Subdomain nutzen. Wenn Sie beispielsweise die Domain „meine-firma.com“ besitzen, können Sie unter jeder Subdomain, z. B. „vcards.meine-firma.com“, auf Ihre vCards zugreifen.

04 Ist die NFC-Karte zwingend erforderlich?

Nein, wir bieten die NFC-Karte als optionalen Kauf an. Die Kosten betragen 7,95 € pro Karte (einmalige Zahlung). Die NFC-Karte ist zwar eine gute Option, Sie können Vcards aber auch nur mit der mobilen Version nutzen.

05 Können für verschiedene Länder oder Teams separate Marken erstellt werden?

Ja, Sie können für jedes Land, jede Region oder jedes Team unterschiedliche Marken erstellen und jedes Team kann seine eigene Marke verwalten.

06 Welche Vorteile haben Vcards von TicTAP gegenüber anderen Lösungen?

Es handelt sich um eine vollständig webbasierte Lösung, für die weder eine Anwendung noch die Unterstützung einer IT-Abteilung erforderlich ist. Sie können Inhalte in Echtzeit bearbeiten, ohne dass Programmierer erforderlich sind, und mehrere Marken und Vorlagen segmentiert verwalten.

07 Können mehrere Visitenkarten gleichzeitig gescannt werden?

Ja, Sie können mehrere Karten mit einem einzigen Foto scannen und die Kontakte in Excel herunterladen.

08 Kann ich die Karte in verschiedenen Sprachen haben?

Ja, die Plattform unterstützt das Bearbeiten von Karten in den angeforderten Sprachen.

09 Kann das Kartendesign geändert werden?
Ja, Sie können das Kartendesign ändern, allerdings gegen Aufpreis, abhängig von Ihren Anforderungen.
